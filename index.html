Actividad 2 - Creación de una página web en GitHub

En esta actividad aprenderá a abrirse una cuenta en GitHub, a utilizar el panel de control de GitHub y a utilizar esta misma cuenta de GitHub para alojar su propia web gratis a través de los servidores de GitHub. Suba una imagen de la web en funcionamiento para que podamos verificar que ha conseguido alojar su web.
